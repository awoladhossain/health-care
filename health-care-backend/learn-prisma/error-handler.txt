Best Practice (TypeScript Safe Way)

 catch (err: unknown) {
  if (err instanceof Error) {
    return res.status(StatusCodes.INTERNAL_SERVER_ERROR).json({
      success: false,
      message: err.message,
      error: err.stack,
    });
  }

  return res.status(StatusCodes.INTERNAL_SERVER_ERROR).json({
    success: false,
    message: "Failed to create Admin",
    error: err,
  });
}


Why?
unknown is the correct type for catch errors.
err instanceof Error ensures you safely access .message .name .stack.
